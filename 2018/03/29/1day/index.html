大家好，我是申浩，这是我第一次打卡~

昨天和今天的学习内容主要有以下两大部分

一.git安装，网上安装，测试是否成功用 黑框命令 git –version

1.配置用户信息
    git config  --global user.name 'zhangsan'

git config  --global  user.email 'xxx@163.com'
2.检查
    git config  --global user.name
3.GIT的基本操作
  • git init 初始化仓库

  • git status 查看当前仓库文件状态

    如果存在红色标记 说明有文件没有被暂存或者提交

  • git add xx.xx 添加单个文件到暂存区

  • git add . 添加所有文件到暂存区
  • git commit -m ‘这是描述信息’ 注意：-m一定不要忘记
  • git checkout 02-git_test.html 可以把暂存区的某个文件取到工作区 注意：谨慎使用
  • git log 查看日志 日志有hash值组成

  • git reset –hard asdfsdf(前七位即可) 可以恢复到相应的版本

    4.配置公钥
    ssh-keygen -t rsa -C? “youremail@example.com“ 配置公钥 在任意文档都可以配，这是和上传某个文件时候的GIT区别。
    然后找到Administrator中的，id_rsa.pub，然后复制器内容
    然后上传到GITHUB的profiles
    在这之前必选先 commit文件，存起来先~
    git remote add origin https://github.com/xiaomoe8/sdf.git
    git push -u origin master

二.首先github 搭建个github pages，详情百度！！！！！

2.安装hexo，要在某个文档里边
hexo的安装环境：node.js   和 git
他俩的检查通过黑框检查即可， 比如 git --v
然后hexo的安装步骤为
1.npm install -g hexo-cli
2.找到文件夹，通过hexo的命令 在本地创建blog代码框架 hexo init   <filename>
3.本地预览一下 hexo server  {注意：Ctrl+C是退出预览} 网址是ocalhost:4000， 《更换皮肤略过》
4.创建md（带有日期标题等等）的命令是    hexo new

###三.发布BLOG
1.首先要安装依赖库
在本地黑框打开输入下边命令
npm install hexo-deployer-git –save
npm install hexo-generator-json-content –save
2.配置git
在当前文件创建git（如输入git init）
然后找到当前博客的_config.yml
把以下信息修改一下
deploy:
type: git
repo: git@github.com:yourname/yourname.github.io.git
branch: master
3创建个public文件（public为镜像文件）
命令框输入：hexo generate
4最后输入 hexo deploy

###最后最后最后，成功！
